PHP echo 和 print 语句

  • PHP echo 和 print 语句

  • print - 只允许输出一个字符串，返回值总为 1

注意！echo 输出的速度比 print 快， echo 没有返回值，print有返回值1。

**echo 和 print 都是一个语言结构，使用的时候可以不用加括号，也可以加上括号： echo 或 echo() print print()**。
